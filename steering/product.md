# Product Overview
<!-- Last rollup: 2026-03-22, 004-table-view-operations -->

## プロダクトの目的

RustyDataBaseNotes は，SaaS に依存せずデータを失わずに使えるローカル完結のノートブックアプリである。
ページ・ブロック・データベース・プロパティ・ビューという中核概念により，
テキストノートと構造化データの両方を一つのアプリケーションで管理できる。

## 対象ユーザー

- ローカルファーストでデータ完全性を重視する個人ユーザー
- テキストノートに加えて，構造化データ（タスク管理，情報整理等）を同一ツールで扱いたいユーザー

## ドメイン境界

| 概念 | 説明 |
|------|------|
| Page | ノートブックの基本単位。タイトル・作成日時・更新日時を持つ。スタンドアロンまたはデータベースに所属 |
| Block | ページに属するテキストコンテンツの最小単位。種別（現在は text のみ）・内容・表示順序を持つ |
| Database | ページ集合に共通のプロパティスキーマを付与する上位概念 |
| Property | データベースに属するスキーマ定義（テキスト・数値・日付・セレクト・チェックボックスの5型） |
| PropertyValue | 特定ページの特定プロパティに対する値（交差エンティティ） |
| View | データベースの表示設定（ソート・フィルタ・グルーピング条件を保持）。1DB=1ビュー（将来は複数ビュー対応予定） |

## 主要ユースケース

<!-- rollup: init, 2026-03-22 -->

- **ページ CRUD**: ページの新規作成・一覧表示・タイトル更新・削除（001-page-persistence）
- **ブロック編集**: ページを開きプレーンテキストブロックを追加・編集・並び替え・削除し，明示的に保存（002-block-editor）
- **データベース管理**: データベースの作成・タイトル変更・削除。ページとの紐づけ管理（003-database-properties）
- **プロパティ定義**: データベースに5種類のプロパティ型を追加・編集・並び替え・削除（003-database-properties）
- **テーブルビュー**: データベース内のページをテーブル形式で一覧表示し，プロパティ値をインライン編集（003-database-properties）
<!-- rollup: 004-table-view-operations, 2026-03-22 -->
- **ソート**: カラムヘッダークリックで単一ソート（昇順→降順→解除），ソートパネルで複数カラムの優先順位付きソート（最大5件）（004-table-view-operations）
- **フィルタ**: プロパティ型に応じた演算子でフィルタ条件を設定し AND 結合で絞り込み（最大20件）。0件時はメッセージと解除手段を提示（004-table-view-operations）
- **グルーピング**: 1つのプロパティ値でグループ化表示。グループヘッダーに件数表示，折りたたみ・展開が可能（004-table-view-operations）
- **ビュー設定永続化**: ソート・フィルタ・グルーピング設定の自動保存・復元。プロパティ削除時の孤立条件自動修復（004-table-view-operations）

## 提供している価値

- 完全ローカル動作 — 外部通信・テレメトリなし，データはユーザーの端末内にのみ存在
- トランザクション保護 — すべての書き込みが原子的で，クラッシュ時にデータ破損しない
- テキストノートと構造化データの統合 — 同一ページがブロック（自由テキスト）とプロパティ（構造化データ）の両方を持てる
- 複数ビューへの拡張性 — テーブルビューを起点に，将来ボード・ガントチャート等に対応可能なモデル設計
